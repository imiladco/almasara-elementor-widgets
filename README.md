# افزونه ویجت‌های الماسارا (Almasara Elementor Widgets)

افزونه مستقل ویجت‌های المنتور برای فروشگاه الماسارا — جدا از پوسته فرزند.

## نصب

این پوشه (`almasara-elementor-widgets`) باید در مسیر زیر روی هاست قرار بگیرد:

```
wp-content/plugins/almasara-elementor-widgets/
```

> توجه: این پوشه داخل ریپوی پوسته نگهداری می‌شود ولی **نباید** داخل پوشه پوسته (`wp-content/themes/almasara/`) آپلود شود.

سپس از پیشخوان وردپرس → افزونه‌ها، افزونه **Almasara Elementor Widgets** را فعال کنید.

## پیش‌نیازها

- PHP 7.4+
- المنتور 3.7 یا بالاتر (با API جدید `elementor/widgets/register` نوشته شده)
- ووکامرس

## ویجت‌ها

### ویژگی‌های محصول الماسارا

در پنل المنتور داخل دسته «الماسارا» (و همچنین دسته ووکامرس) در دسترس است.

**تب محتوا:**
- سطر اول: عنوان (با تگ قابل انتخاب) + خط جداکننده + آیکون
- آیکون: از کتابخانه آیکون المنتور (شامل آپلود SVG) یا آپلود تصویر با هر فرمت؛ SVG به‌صورت inline رندر می‌شود تا کاملاً قابل استایل‌دهی باشد
- آیتم‌ها (ریپیتر): هر آیتم می‌تواند ویژگی محصول (سراسری یا سفارشی)، دسته‌بندی، برچسب، SKU، امتیاز خریداران، وضعیت موجودی، وزن، ابعاد یا یک متافیلد دلخواه باشد — مقدار به‌صورت داینامیک از محصول جاری خوانده می‌شود
- لینک: انتخاب اینکه آیتم‌ها، عنوان، کل سطر اول یا کل ویجت لینک شوند؛ برای آیتم‌ها می‌توان یک لینک یکسان یا لینک جدا برای هر آیتم تعریف کرد
- پیش‌نمایش ادیتور: خارج از صفحه محصول، با شناسه محصول دلخواه یا آخرین محصول

**تب استایل:**
- چیدمان آیتم‌ها: فلکس یا گرید با همه امکانات (ستون‌ها، قالب سفارشی گرید، جهت، wrap، justify/align، gap سطر و ستون، عرض آیتم)
- استایل کامل سطر اول، عنوان، خط جداکننده، آیکون، باکس آیتم‌ها، عنوان و مقدار هر آیتم
- حالت هاور برای همه بخش‌ها (رنگ، پس‌زمینه، کادر، سایه، جابه‌جایی، چرخش آیکون، مدت انیمیشن)
- رنگ‌آمیزی هوشمند SVG: حالت «هوشمند» فقط صفت‌هایی که در فایل رنگ دارند (fill یا stroke) را بازرنگ می‌کند تا آیکون خراب نشود؛ حالت‌های اجباری fill / stroke و حالت حفظ رنگ اصلی هم موجود است

## نکته امنیتی SVG

فایل‌های SVG هنگام رندر inline پاک‌سازی می‌شوند (حذف اسکریپت، رویدادها و لینک‌های خطرناک). برای فعال شدن آپلود SVG در کتابخانه رسانه، از تنظیمات المنتور گزینه «Unfiltered File Uploads» را فعال کنید یا از افزونه‌ای مانند Safe SVG استفاده کنید.
