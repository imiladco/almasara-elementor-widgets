# افزونه ویجت‌های الماسارا (Almasara Elementor Widgets)

افزونه مستقل ویجت‌های المنتور برای فروشگاه الماسارا — جدا از پوسته فرزند.

## نصب

این مخزن باید در مسیر زیر روی هاست قرار بگیرد:

```
wp-content/plugins/almasara-elementor-widgets/
```

سپس از پیشخوان وردپرس → افزونه‌ها، افزونه **Almasara Elementor Widgets** را فعال کنید.

## پیش‌نیازها

- PHP 7.4+
- المنتور 3.13 یا بالاتر
- ووکامرس

## ویجت‌ها

### ویژگی‌های محصول الماسارا

در پنل المنتور داخل دسته «الماسارا» (و دسته ووکامرس) در دسترس است.

**تب محتوا:**

1. **معرفی** — عنوان، تگ، سوئیچ خط جداکننده، و آیکون (کنترل رسانه با دو گزینه «انتخاب تصویر» و «آپلود SVG»؛ فایل SVG به‌صورت inline و پاک‌سازی‌شده رندر می‌شود)
2. **چیدمان** — کنترل‌های استاندارد فلکس/گرید (مثل کانتینر المنتور) جداگانه برای سطر معرفی و سطر آیتم‌ها: جهت، تراز کردن محتوا، تراز موارد، شکاف‌های لینک‌شده (سطر/ستون)، Wrap؛ و در حالت گرید: ستون‌ها و ردیف‌ها (fr)، قالب ستون سفارشی و جهت چینش
3. **ویژگی‌ها** — «تعداد آیتم‌های نمایشی» + ریپیتر آیتم‌های انتخابی (ویژگی سراسری/سفارشی، دسته‌بندی، برچسب، SKU، امتیاز، موجودی، وزن، ابعاد، متافیلد). اگر محصول آیتم‌های انتخابی را نداشت، به‌طور خودکار از سایر ویژگی‌های خودِ محصول تا رسیدن به تعداد تعیین‌شده تکمیل می‌شود. مخفی‌سازی آیتم‌های خالی، جداکننده مقادیر چندتایی و حداکثر تعداد مقادیر هم اینجاست.
4. **لینک** — انتخاب هدف لینک: آیتم‌ها (لینک یکسان یا جدا برای هر آیتم)، عنوان، کل سطر معرفی، یا کل ویجت

**تب استایل:**

1. **معرفی** — تایپوگرافی عنوان، نوع و ضخامت خط جداکننده (عرض همیشه خودکار است)، سایز و پدینگ آیکون؛ سپس دو تب «عادی / هاور» که همه رنگ‌های سطر معرفی را یکجا دارد: رنگ عنوان، رنگ خط جداکننده، رنگ آیکون (fill) و رنگ خطوط آیکون (stroke)
2. **آیتم** — دو تب «عنوان / مقدار» (تایپوگرافی، رنگ، سایه متن، رنگ هاور)؛ چیدمان داخلی و پدینگ باکس؛ دو تب «عادی / هاور» (پس‌زمینه، حاشیه، رادیوس، سایه)؛ و مدت‌زمان انیمیشن

**رنگ‌آمیزی خودکار SVG:** کنترل «رنگ آیکون» فقط بخش‌هایی از SVG را بازرنگ می‌کند که در فایل fill دارند (یا بدون رنگ‌اند و به fill پیش‌فرض تکیه کرده‌اند) و «رنگ خطوط آیکون» فقط بخش‌های دارای stroke را. هر کدام خالی بماند، رنگ اصلی همان بخش حفظ می‌شود — بدون نیاز به انتخاب دستی حالت.

## نکته امنیتی SVG

فایل‌های SVG هنگام رندر inline پاک‌سازی می‌شوند (حذف اسکریپت، رویدادها و لینک‌های خطرناک). برای فعال شدن آپلود SVG در کتابخانه رسانه، از تنظیمات المنتور گزینه «Unfiltered File Uploads» را فعال کنید یا از افزونه‌ای مانند Safe SVG استفاده کنید.
